# devtools-final
# วิธีการใช้งาน

1. ทำการ clone โปรเจค
```
git clone https://github.com/it62070041/Devtool-Final.git
```
2. เข้าไปใน folder โปรเจค
```
cd Devtool-Final
```
3. ทำการ build image and deploy
```
docker compose up -d
```
4. backend 
```
http://localhost:3013
```
ดูข้อมูลอื่นๆได้ใน [wiki](https://github.com/it62070041/Devtool-Final/wiki)
